```
# 默认为1m
client_max_body_size 1024m;
```


### nginx重启几种方法

平滑重启命令：
```
ps -ef|grep nginx

kill -HUP 住进称号或进程号文件路径
```

或者使用

```
/usr/nginx/sbin/nginx -s reload
```


注意，修改了配置文件后最好先检查一下修改过的配置文件是否正 确，以免重启后Nginx出现错误影响服务器稳定运行。

判断Nginx配置是否正确命令如下：
```
/usr/nginx/sbin/nginx -t
```
